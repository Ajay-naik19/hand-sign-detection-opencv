# Hand Sign Detection using OpenCV

This mini project demonstrates a real-time **Hand Sign Detection System** using **Python**, **OpenCV**, and **MediaPipe**.  
The system tracks the hand through the webcam and identifies different hand signs using image processing and machine learning techniques.
